KJ-Player
這是一個為9779s打造的個人化音樂應援播放器。使用者可以透過此網頁瀏覽歌曲清單、切換播放、查看同步歌詞及雙語翻譯。

#專案功能
1.音樂庫播放：支援多個歌單分類。
2.同步歌詞：提供即時滾動歌詞顯示與中/泰雙語切換功能。
3.支援行動裝置與電腦瀏覽器，提供良好的使用者體驗。
4.PWA 支援

#使用技術
前端：HTML5, CSS3, JavaScript 
影音處理：HLS.js
架構：PWA
