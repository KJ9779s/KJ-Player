# KJ-Player

KJ-Player 是一個為 9779s 打造的個人化音樂應援播放器。使用者可以透過此網頁瀏覽歌曲清單、切換播放、查看同步歌詞及雙語翻譯。

---

## 功能 / Features

1. **音樂庫播放**：支援多個歌單分類。
2. **同步歌詞**：提供即時滾動歌詞顯示與中/泰雙語切換功能。
3. **響應式設計**：支援行動裝置與電腦瀏覽器，提供良好的使用者體驗。
4. **PWA 支援**：可離線存取，提供如同原生 App 的操作體驗。

---

## 使用技術 / Tech Stack

* **前端**：HTML5, CSS3, JavaScript
* **影音處理**：HLS.js
* **架構**：PWA (Progressive Web App)

---

## 安裝與說明 / Installation

* **瀏覽器存取**：透過支援 HTTPS 的網頁瀏覽器開啟網址。
* **加入桌面**：在瀏覽器選單中選擇「加入主畫面」，即可快速開啟。
